# Contributing

This repository accepts corrections to the public API contract, dependency-free examples, and documentation.

Before opening a pull request:

1. Run `npm test`.
2. Do not add credentials, private URLs, captured customer content, or generated artifacts.
3. Keep examples on the fixed production endpoint and read the key only from `LATCHSHOT_API_KEY`.
4. Keep the hosted-service source and proprietary runtime outside this repository.

For service behavior or feature requests, open an issue first and describe the public-page job and expected monthly volume. Private/authenticated browser automation remains out of scope.
